# chn-h5-lab

## 项目简介

`chn-h5-lab` 是一个用于归档和展示多个独立 HTML 页面的仓库。本仓库不针对任何特定网站或域名，主要用于存放和发布各种实验性、示例性或静态 HTML 页面，方便通过 GitHub Pages 直接访问和分享。

## 目录结构

```
.
├── index.html          # 仓库主页，用于导航到各个页面
├── pages/              # 存放各个独立 HTML 页面的目录
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
├── assets/             # 公共资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `index.html`：仓库默认页面，通常包含页面列表或导航链接。
- `pages/`：所有独立 HTML 页面均放置在此目录下，按需命名。
- `assets/`：存放页面共享的样式、脚本、图片等资源文件。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，每个页面可单独通过 GitHub Pages 访问。页面内容可能涉及技术演示、原型展示、笔记整理等，不构成任何商业或推广用途。

- 页面命名尽量简洁明了，便于识别。
- 页面之间不互相依赖，均可独立运行。
- 资源文件尽量使用相对路径，确保页面在本地或线上均可正常显示。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/chn-h5-lab.git
   ```
2. 直接在浏览器打开 `index.html` 或 `pages/` 下的任意页面即可查看。
3. 若启用 GitHub Pages，可通过相应 URL 访问在线版本。
